## Ingredients

After the installation finished, you'll get 5 items:

```ts
import {
  initStore,
  createStore,
  createStores,
  createQuery,
  createMutation,
} from 'floppy-disk'
```

Here is the structure of dependencies of each items:

The `createQuery` function was built on top of `createStores`.
So, all the things we can do in `createStores`, we can do it on `createQuery` too.
Therefore, before jump into query & mutation, it is better to understand how store & stores work.
